# LSO Bar Exam Hub

4-exam practice hub for the Ontario Bar Exam:
- LSO Barrister Practice Exam A (160 questions)
- LSO Solicitor Practice Exam A (160 questions)  
- AccessBarPrep Barrister Mini Exam (80 questions)
- AccessBarPrep Full Barrister Exam (160 questions)

## Deploy

This is a static single-file app served via NGINX on Railway.

Push to GitHub → connect to Railway → get a public URL.
